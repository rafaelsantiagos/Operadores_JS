Operadores do JavaScript

Operadores aritméticos

+ = Adição; EXEMPLO: 5+2=7

- = Subtração; EXEMPLO: 5-2=3

* = Multiplicação; EXEMPLO: 5*2=10

/ = Divisão; EXEMPLO: 5/2=2.5

% = Módulo (resto da divisão); EXEMPLO: 5%2=1

** = Potência; EXEMPLO: 5**2=25

CUIDADO COM A ORDEM DE OPERAÇÕES ! ! !

Operadores de atribuição

Simples

x = a + b;Qualquer variável recebendo o valor da operação

Auto-Atribuição

n += a = Recebe o próprio valor +a (n = n + a);

n -= a = Recebe o próprio valor -a (n = n -a);

n *= a = Recebe o próprio valor *a (n = n * a);

n /= a = Recebe o próprio valor /a (n = n / a);

n **= a = Recebe o próprio valor **a (n = n ** a);

n %= a = Recebe o próprio valor %a (n = n % a);

Incremento

x++ (x = x + 1) e x-- (x = x - 1)

Operadores relacionais

> < >= <= == !=

Os resultados disto sempre serão valores booleanos (true or false)

n == 'n', para o JavaScript é true. Para tornar false é só usar o operador de identidade. n === 'n' é false, pois === vê o tipo (number ou string)
